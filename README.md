# Android Port Requirements for Vita SO Loader 

## Requirements Summary
✔️ ARMv6 OR ARMv7 exexutable\
✔️ OpenGL: GLES 1 or GLES 2\
✔️ FMOD: libfmod.so\
🔶 OpenGL: GLES 3+\
🔶 FMOD: libfmodevent.so, libfmodex.so, libfmodstudio.so\
❌ ARMv8 executable only\
❌ Kotlin only\
❌ Unity games\
❌ Java games\
